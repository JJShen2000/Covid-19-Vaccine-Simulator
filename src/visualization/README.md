# README
## Required packages
- dash<br>
  `pip install dash`
-  plotly<br>
  `pip install plotly`
  
## Usage
### Update data
Assume data dir is in models/sim_data
`python ./init_data.py`
### Run
`python ./app.py`

The graph will be running on http://127.0.0.1:8050/.
