# Covid-19-Vaccine-Simulator
This is Covid-19 Vaccine Simulator!

## Usage

```
./run [-cCd]
./run [num_of_sim_rounds]
```

- -c : Clean result of simulation
- -C : Clean all data
- -d : Draw line charts of the result
